
# Limited Intercom

<kbd>  admintoolbox_intercomlock  </kbd>  
<kbd>  Boolean  ➞  False  </kbd>

<br>
<br>

## Description

*Locks the intercom for non-whitelisted players.*

<br>
<br>

## Example

*Players will not be locked out by the whitelist.*

```yaml
admintoolbox_intercomlock : false
```
