
# Join Name

<br>

## Description

*Displays a players name upon joining.*

<br>

*Ignored if  `admintoolbox_player_` <br>
`join_info_extended`  is enabled.*

<br>
<br>

## Example

*Joining players names will not be shown.*

```yaml

```
