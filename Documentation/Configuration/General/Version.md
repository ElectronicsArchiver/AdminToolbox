
# Version Checking

<kbd>  atb_disable_networking  </kbd>  
<kbd>  Boolean  ➞  False  </kbd>

<br>
<br>

## Description

*Disables checking for the newest version via **GitHub**.*

<br>
<br>

## Example

*Will not search for the newest version of **AdminToolbox**.*

```yaml
atb_disable_networking : true
```
