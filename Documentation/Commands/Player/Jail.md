
# Jail

<kbd> Jail  </kbd>

<br>
<br>

## Description

*Jails the given player for the given amount of time.*

<br>
<br>

## Un-jailing

*Simply reuse the command on the jailed player.*

<br>
<br>

## Examples

*Jails `Mike` for `1 Year`.*

```shell
JAIL Mike
```

<br>

*Jails `Mike` for `x`.*

```shell
ATSPEAK Mike x
```
