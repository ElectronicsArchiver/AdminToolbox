
# Narrator

<kbd>  ATSPEAK  </kbd>   or   <kbd>  ATINTERCOM  </kbd>   or   <kbd>  AT-SPEAK  </kbd>

<br>
<br>

## Description

*Set the given player as the intercom speaker.*

<br>
<br>

## Note

*Cancels any ongoing broadcasts.*

<br>
<br>

## Examples

*Makes  `Mike`  the narrator.*

```shell
ATSPEAK Mike
```

<br>

*Makes **yourself** the narrator.*

```shell
ATSPEAK
```

<br>
