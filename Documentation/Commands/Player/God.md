
# GodMode

<kbd>  ATGOD  </kbd>

<br>
<br>

## Description

*Sets the given player into GodMode.*

<br>
<br>

## No Damage

Append  `nodmg`  to simultaneously activate it.

<br>
<br>

## Examples

*Makes  `Mike`  a god with NoDmg.*

```shell
ATGOD Mike true nodmg
```

<br>

*Toggles whether or not  `Mike`  is a god.*

```shell
ATGOD Mike
```
